# Fies Store

A small web app so my friends can manage inventory at Anime cons.

## Constraints

 * No internet
 * Windows Seven
 * Work in Cash

## Wants

 * Item / inventory tracking
   * item type
   * item description
   * creator
   * suggested price
   * sale price
   * qty?
 * Commision tracking
   * overnight or mail order
   * email
   * mailing address
